# JSTools JSON Studio

## Tagline

A private, lossless desktop workspace for large JSON — explore, search, query, and diff files up to 500 MB, entirely on your machine.

## One-liner

JSON Studio is a fast, offline desktop app for reading and working with JSON and JSONL files of any size — without ever uploading your data anywhere.

## Short description (for a store page or repo header)

JSON Studio is a desktop application for developers who work with large or sensitive JSON. It opens files up to 500 MB instantly, preserves your data exactly — big integers, high-precision decimals, key order and the original bytes are never altered — and gives you a virtualized tree, raw source view, full-text search, JSONPath queries, structural diff, and a JSONL table, all running 100% locally with no network access, accounts, or telemetry.

## Full description (for a landing page)

**JSON Studio is a private, lossless workspace for JSON.**

Most JSON tools either choke on big files, silently mangle your numbers, or send your data to a server. JSON Studio does none of that. It's a native desktop app built for speed and correctness: open a half-gigabyte file and start navigating immediately, knowing that what you see is byte-for-byte what's on disk.

Everything runs on your own machine. There are no accounts, no uploads, no analytics, and no network calls — making it safe for proprietary, regulated, or personal data.

### What you can do

- **Browse huge files smoothly.** A virtualized tree and raw view stay responsive on documents up to 500 MB, with collapse/expand, expand-to-depth, type and item-count badges, and full keyboard navigation.
- **Trust your data.** Big integers, high-precision decimals, key order, and the exact original text are preserved losslessly — nothing is rounded, reordered, or reformatted.
- **Find anything fast.** Search across keys and values with case sensitivity, whole-field matching, regular expressions, and subtree scoping — then jump straight to a match in the tree.
- **Query with JSONPath.** Run RFC 9535 JSONPath expressions with per-document history, execution timing, result counts, and export to JSON or JSONL.
- **Compare two documents.** A structural diff (key order ignored) matches arrays by index or by a key field and highlights what was added, removed, or changed.
- **Work with JSONL / NDJSON.** View line-delimited records as a virtualized table with selectable columns, field statistics, and invalid-row highlighting.
- **Preview embedded images.** String values that are `data:image/…` URIs or base64 (PNG, JPEG, GIF, WebP, SVG) can be viewed inline — handy for avatars and thumbnails stored in JSON.
- **Format and export.** Copy pretty-printed or minified JSON, or save the re-serialized document to a file in one click.
- **Paste and scratch.** Paste JSON straight into a new tab, edit it in place, compare two pasted payloads, and save when ready.
- **Stay in flow.** Multiple documents in tabs, remembered sessions, live reload when a file changes on disk, a node detail panel with copyable path and value, and themes (System / Light / Dark).

### Why people use it

- **Private by design** — your data never leaves your computer.
- **Lossless** — exact fidelity for numbers, ordering, and formatting.
- **Built for scale** — large files open and scroll without freezing.
- **Free and open source.**
